## **Como Rodar o Projeto**

### Requisitos:
- Docker
- Docker Compose

### Passos:
1. Clone o repositório:
   ```bash
   git clone https://github.com/MatheusSan99/StarWars.git
   ```
2. Suba o ambiente:
   ```bash
   docker build --no-cache -t starwars .
   ```
3. Acesse a aplicação no navegador:
   ```
   http://localhost
   ```
